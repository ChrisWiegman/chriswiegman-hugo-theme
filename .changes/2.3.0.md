## 2.3.0 - 2026-03-29

### Features

* Generate WebP srcset for inline content images via render-image.html render hook
* Add theme-color meta tag for mobile browser chrome styling
* Add loading=lazy to all content images for deferred off-screen image loading

### Bug Fixes

* Remove deprecated call to .Site.AllPages 
* Add itemprop=datePublished to time element to correctly associate date with BlogPosting schema microdata
* Remove fetchpriority=high from content images in render-image.html; reserved for LCP elements only
* Add datetime attribute to time element on single posts for correct semantic HTML and microformat support
