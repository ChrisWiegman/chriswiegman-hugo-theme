## 2.5.0 - 2026-05-03

### Features

* RSS dc:creator field is now configurable via the params.author site setting instead of being hardcoded
* Added an `asset-image` shortcode for better handling of images in html files.

### Bug Fixes

* Remove monospace font from blockquotes so prose quotations no longer render like code blocks
* Fix responsive image breakpoints so mobile devices receive the correct image size instead of always loading the full 850px image
* Fix template build error when imageSizes config values are TOML integers

### Chores

* Replace Prettier with the official Hugo gotmplfmt formatter for Go template files
* Migrate dev site configuration from hugo.json to hugo.toml
